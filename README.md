# OIBSIP
"A curated repository of high-performance web apps, including a portfolio and thermal converter. Developed for the Oasis Infobyte Internship Program, this project showcases clean code, intuitive UX, and a passion for elegant web engineering."

This repository contains all the projects completed during my Oasis Infobyte Web Development and Designing Internship.

During this internship, I developed responsive and interactive web applications using HTML, CSS, and JavaScript. These projects helped me improve my frontend development skills and gain practical experience in building user-friendly websites.

## Level 1 Tasks

### Task 1 - Landing Page

A responsive and modern landing page designed using HTML and CSS.  
The project focuses on creating an attractive user interface with proper layout, styling, and responsiveness.

### Task 2 - Portfolio Website

A personal portfolio website showcasing skills, projects, and contact information.  
It represents my profile and frontend development abilities with a clean and professional design.

### Task 3 - Temperature Converter

A temperature conversion web application developed using HTML, CSS, and JavaScript.  
It allows users to convert temperature values between different units with an interactive interface.

## Technologies Used

- HTML5
- CSS3
- JavaScript
